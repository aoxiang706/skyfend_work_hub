# 示例笔记

这是一条示例笔记，用于演示**站内阅读**效果。

- 将你的 `.md` 文件放到 `work-hub/public/content/notes/` 目录
- 在 `public/content/notes/index.json` 的 `items` 里添加一条记录，包含：`id`、`title`、`file`、`category`（分类）、可选 `date`、`excerpt`（摘要）
- 推送后即可在「笔记资料」中按分类浏览，点击后在当前站点内打开阅读，无需跳转 GitHub

## 分类建议

- **项目文档**：sss_ptz100、ptz100_agx 等工程说明
- **设备/SDK**：和普 PTZ、耐杰 PTZ 等摘要
- **工作笔记**：日常总结、会议记录等
